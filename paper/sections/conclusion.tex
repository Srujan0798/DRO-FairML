We introduced FairnessTargetedPGD, the first gradient-based adversarial attack targeting fairness metrics. Our evaluation on 285 experiments reveals that DRO-FAIR's robustness is \textbf{not universal}: it defends against IF attacks on tabular data (64--97\% reduction, $p < 0.05$) but \textbf{inverts} on image features---producing worse fairness than naive training. This modality-dependent behavior reveals that DRO requires feature-demography correlation to be effective. This finding cautions against deploying DRO-based fair ML on transfer-learned image features without careful validation.
