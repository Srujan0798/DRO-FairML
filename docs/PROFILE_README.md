## Featured Project — DRO-FAIR

Distributionally robust fair classification under **adversarial data corruption**.

- **Algorithm**: Min-max Lagrangian (Algorithm 1, ICML 2026) with corruption-calibrated TV uncertainty sets
- **Key contribution**: Replaced the paper's random noise with adversarial attacks (PGD + coordinated label flips)
- **Results**: Up to **92% DP reduction** on Credit, **100%** on LSAC across 150 experiments
- **Stack**: PyTorch · NumPy · scikit-learn · CI tested on Python 3.10–3.12
